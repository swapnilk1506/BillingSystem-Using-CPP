# BillingSystem-Using-CPP
This is a basic console-based Billing System developed using C++ and Object-Oriented Programming (OOP) concepts. The project allows users to perform CRUD operations (Create, Read, Update, Delete) on billing records in a simple and beginner-friendly way.  This project is designed to strengthen fundamentals of C++, OOP, and STL vector usage.
# Technologies Used
C++
Object-Oriented Programming (OOP)
STL (vector)
Console Application
# Features
Add new bill items
View all bill records
Update existing bill details using Item ID
Delete bill records using Item ID
Menu-driven console interface
